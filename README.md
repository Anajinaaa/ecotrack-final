# ecotrack-final
EcoTrack 🌿
Gannon University Senior Design Project - Team C

EcoTrack is a Progressive Web App (PWA) designed to encourage sustainable living by tracking daily activities, calculating carbon offsets (Eco-Points), and fostering a competitive community leaderboard.

🚀 Quick Start (Docker)
You do not need to install Python or Node.js on your local machine. Everything is containerized.

Clone the repository:

Bash
git clone https://github.com/Anajinaaa/ecotrack-final.git
cd ecotrack-final
Launch the application:

Bash
docker-compose up --build
Access the services:

Frontend: http://localhost:3000

Backend API (Swagger): http://localhost:8000/docs

🏗️ Technical Architecture
Frontend: Next.js (JavaScript & Tailwind CSS)

Backend: FastAPI (Python 3.11)

Infrastructure: Docker & Docker Compose
